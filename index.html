Welcome to Weather Forecast Website

This weather Website is a web-based system that allows users to search for current weather information as well as weather forecast of different cities around the world. Users can search for weather information by city name. This weather app uses the openweathermap API which is found at www.openweathermap.org

Use openweathermap API to get weather information of any city.
